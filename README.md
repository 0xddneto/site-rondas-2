# SITE RONDESP RECÔNCAVO - INSTRUÇÕES

## Arquivos Criados

1. **index.html** - Página principal com informações sobre a RONDESP Recôncavo
2. **pelotao3.html** - Página especial do 3º Pelotão com destaque para o Tenente Maabreh

## Como Visualizar

### Opção 1: Abrir diretamente
- Clique duas vezes no arquivo `index.html` para abrir a página principal
- Use o botão "CLIQUE PARA VER O MELHOR PELOTÃO" para navegar até a página do 3º Pelotão

### Opção 2: Servidor local
Para melhor visualização com todas as imagens carregando corretamente, recomenda-se usar um servidor local:

```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Depois acesse: http://localhost:8000/index.html

## Estrutura do Site

### Página Principal (index.html)
- **Header**: Brasão da RONDESP Recôncavo
- **Seção Fundação e História**: Informações sobre a criação e desenvolvimento da unidade
- **Seção Princípios**: Valores fundamentais da corporação  
- **Seção Valores e Missão**: Missão e visão institucional
- **Seção Estrutura**: Equipamentos e parcerias operacionais
- **Estatísticas**: Resultados operacionais da unidade
- **Botão de navegação**: Link para página do 3º Pelotão

### Página do 3º Pelotão (pelotao3.html)
- **Destaque Especial**: Tenente Maabreh - Comandante do 3º Pelotão
- **Galeria de Fotos**: Imagens da equipe em ação
- **Valores do Pelotão**: Força, Honra, União e Coragem
- **Botão de retorno**: Volta para página principal
